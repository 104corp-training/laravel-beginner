# Laravel beginner

## 運行環境需求

參照官方文件 [Laravel 6 requirements](https://laravel.com/docs/6.x#server-requirements) 可達最低運行需求。

## 必要工具

- Docker
- Docker Compose

## API
-   API文件
```
/api/documentation
```
-   所有課程的API
```
/api/courses
```
-   課程的詳細資料的API
```
/api/courses
```


## 頁面資料
-   所有課程的頁面資料
```
/courses
```
-   課程的詳細資料
```
/course/{id}
```
